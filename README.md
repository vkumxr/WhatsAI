WhatsAI is an open-source WhatsApp automation workflow built with n8n and the WhatsApp Cloud API. 
This project enables AI-driven message sending, automated workflows, chatbot logic, and full integration 
with LLMs like GPT, Gemini, and LLaMA.

Includes example workflows, setup guides, and integration notes for creating your own WhatsApp AI assistant.
Ideal for automation engineers, developers, and anyone building WhatsApp-based AI bots.
